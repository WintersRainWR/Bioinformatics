## 一、RNA-seq 中基因表达量归一化方法

在 RNA-seq 数据分析中，直接使用原始 reads 计数来比较基因表达是不合理的，因为测序深度和基因长度都会对结果产生影响。因此，需要对数据进行归一化处理，使不同样本之间具有可比性。常见的归一化方法主要包括以下几种：

### 1. RPKM（Reads Per Kilobase of transcript per Million mapped reads）

RPKM 是最早提出的一种归一化方法，主要用于单端测序数据。它同时校正了测序深度与基因长度，适用于同一样本中不同基因之间的比较，但是不适合跨样本比较（存在组成偏差问题）

### 2. FPKM（Fragments Per Kilobase of transcript per Million mapped reads）

FPKM 是 RPKM 的扩展版本，主要用于双端测序。其核心思想与 RPKM 相同，只是将 reads 替换为 fragment，更适合 paired-end 数据，但是仍然存在样本间比较不稳定的问题

### 3. TPM（Transcripts Per Million）

TPM 是目前更常用的一种归一化方法，对 RPKM 进行了改进。其核心思想是先对基因长度进行校正，再进行测序深度归一化。

计算步骤如下：

1. 计算每个基因的 reads / gene length  
2. 对所有基因的该值求和  
3. 每个基因除以总和，并乘以 \(10^6\)

其特点为：不同样本之间的总表达量相同，更适合跨样本比较

### 4. CPM（Counts Per Million）

CPM 仅对测序深度进行归一化，不考虑基因长度，常用于差异表达分析的预处理（如 edgeR），更适用于同一基因在不同样本中的比较

### 5. TMM（Trimmed Mean of M-values）

TMM 是一种更为复杂的归一化方法，主要用于消除样本间的组成偏差，常见于 edgeR 工具中。

基本思想是选择一个参考样本，计算各基因在不同样本中的表达比值，去除极端值后求加权平均。专门用于跨样本比较，能有效处理高表达基因带来的偏差，常用于差异表达分析

## 四、tumor-transcriptome-demo.tar.gz提供了结肠癌(COAD)，直肠癌(READ)和食道癌(ESCA)三种癌症各50个样本的bam文件用featureCount计算产生的结果。请大家编写脚本将这些文件中的counts合并到一个矩阵中(行为基因，列为样本), 计算logCPM的Z-score，并用heatmap展示，提供代码和heatmap。根据heatmap可视化的结果，你认为这三种癌症中哪两种癌症的转录组是最相似的?

写出脚本：

```
read_counts_matrix <- function(dir_path){
  files <- list.files(path = dir_path, full.names = TRUE)
  data_list <- lapply(files, function(f){
    df <- read.table(f, header = TRUE)[, c(1,7)]
    colnames(df) <- c("Geneid", basename(f))
    return(df)
  })
  mat <- Reduce(function(x, y) merge(x, y, by = "Geneid", all = TRUE), data_list)
  rownames(mat) <- mat$Geneid
  mat <- mat[,-1]
  return(mat)
}

calc_zscore_matrix <- function(count_mat){
  cpm <- t(1e6 * t(count_mat) / colSums(count_mat))
  log_mat <- log10(cpm + 1)
  z <- (log_mat - rowMeans(log_mat)) / apply(log_mat, 1, sd)
  z <- na.omit(z)
  z <- pmin(pmax(z, -1.5), 1.5)
  return(z)
}

COAD_tot <- read_counts_matrix('./COAD/')
ESCA_tot <- read_counts_matrix('./ESCA/')
READ_tot <- read_counts_matrix('./READ/')

COAD_z <- calc_zscore_matrix(COAD_tot)
ESCA_z <- calc_zscore_matrix(ESCA_tot)
READ_z <- calc_zscore_matrix(READ_tot)

library(pheatmap)

plot_heatmap <- function(mat, title){
  pheatmap(as.matrix(mat),
           cluster_rows = TRUE,
           show_rownames = FALSE,
           show_colnames = FALSE,
           main = title)
}

plot_heatmap(COAD_z, "COAD")
plot_heatmap(ESCA_z, "ESCA")
plot_heatmap(READ_z, "READ")
```
